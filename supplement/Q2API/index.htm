Recommending APIs for API Related Questions in Stack Overflow

The Q2API Corpus:

This page provides the corpus towards the task of Question to API (Q2API) Recommendation. The aim of Q2API is to recommend correct APIs for API related questions in question and answer forums like Stack Overflow. Addressing the Q2API task could speed up the resolution of new questions, thus boosting developers' productivities. This corpus contains 1,234 API related questions with their correct APIs from Stack Overflow. It can be served as a benchmark corpus, and other researchers can verify their approaches addressing the Q2API task over it. We hope that this corpus can be beneficial to other researches and promote the developement of Q2API.

The corpus can be downloaded here: corpus.
